### Para rodar o projeto

São pré-requisitos:
- NodeJS
- npm
- git
- Terminal Bash

```bash
git clone https://github.com/YoYolops/hemakes-tech-challenge &&
cd hemakes-tech-challenge &&
npm i &&
npm run build &&
npm start
```
